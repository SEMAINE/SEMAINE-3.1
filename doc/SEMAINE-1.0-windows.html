SEMAINE-1.0-windows: The Embodied Conversational Agent components of the SEMAINE-1.0 system

Download and installation

Get SEMAINE-1.0-windows.zip (~70mb) from http://sourceforge.net/projects/semaine/ . Unzip it in your preferred location.

The Windows code should run Windows XP and Vista.

If you intend to compile the windows code from source, see SEMAINE-1.0-Dependencies#Windows.

Running the demonstrator

The Windows code can be used in the following demonstrator configuration.

  • SEMAINE-1.0-speech2face
